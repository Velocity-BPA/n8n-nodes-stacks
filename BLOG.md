# Automate Stacks Blockchain Operations with Our New n8n Community Node

We're excited to announce the release of n8n-nodes-stacks, a powerful new community node that brings Stacks blockchain automation to n8n workflows. Created by Velocity BPA, this node enables developers and businesses to seamlessly integrate Stacks blockchain operations into their automation pipelines.

## The Challenge

As blockchain technology matures, developers increasingly need to automate interactions with decentralized networks. The Stacks blockchain, with its unique Bitcoin-anchored smart contracts, STX token operations, NFTs, sBTC, Bitcoin Naming System (BNS), and stacking rewards, presents tremendous opportunities. However, integrating these operations into business workflows has traditionally required extensive custom development and blockchain expertise.

## Meet n8n-nodes-stacks

Our new community node bridges the gap between Stacks blockchain and n8n's visual automation platform. Whether you're building NFT minting workflows, automating stacking operations, managing smart contract deployments, or integrating BNS lookups into your applications, this node provides the building blocks you need.

## Key Features

The n8n-nodes-stacks node supports comprehensive Stacks blockchain operations including:

- **STX Token Operations**: Send, receive, and query STX token transactions
- **Smart Contract Interactions**: Deploy and call Clarity smart contracts directly from your workflows
- **NFT Management**: Mint, transfer, and query NFT collections on Stacks
- **sBTC Integration**: Work with sBTC, the decentralized Bitcoin peg on Stacks
- **Bitcoin Naming System**: Resolve and manage BNS names programmatically
- **Stacking Automation**: Monitor and manage STX stacking rewards

By combining these capabilities with n8n's 400+ existing integrations, you can create powerful workflows that connect blockchain operations with traditional business systems, databases, APIs, and communication tools.

## Getting Started

Installing the node is straightforward. In your n8n instance, run:


npm install n8n-nodes-stacks


After restarting n8n, the Stacks node will appear in your node palette, ready to use in your workflows.
